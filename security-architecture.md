# Truoux Exchange Security Architecture

Security is one of the most critical components of any
cryptocurrency trading platform.

Truoux deploys a multi-layer security architecture.

## Account Security

• Two-factor authentication  
• Permission management  
• Login behavior monitoring  

## Asset Security

The platform uses a **cold and hot wallet separation model**.

- majority of assets stored in cold wallets
- multi-signature authorization
- strict transfer verification

## System Security

Infrastructure protection includes:

- DDoS mitigation
- intrusion detection
- real-time monitoring
- cross-region disaster recovery

These mechanisms provide defense against both
external cyber threats and internal operational risks.
